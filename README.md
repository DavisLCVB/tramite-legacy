# tramite-legacy

Sistema legacy de mesa de partes para laboratorio de auditoría.

## Ejecución

```bash
docker compose up --build
```

Acceso: `http://localhost:8080`

## Diseño compartido

Este módulo replica la guía visual común definida en:

`../style.md`

## Funcionalidades principales

- Registro de documentos por dependencia.
- Carga de archivos de sustento.
- Diagnóstico de conectividad interna.
